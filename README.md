## Descripción

Este proyecto fue desarrollado para la materia Minería de Datos y tiene como objetivo analizar factores asociados a la diabetes mediante técnicas de exploración y preparación de datos.

El análisis se realizó utilizando un dataset basado en encuestas de salud pública del sistema BRFSS (Behavioral Risk Factor Surveillance System) de los Estados Unidos, disponible en Kaggle.

Durante el proyecto se trabajó sobre distintas etapas del proceso de minería de datos, incluyendo:

- Comprensión del problema
- Exploración inicial del dataset
- Análisis de calidad de los datos
- Tratamiento de outliers
- Transformaciones de variables
- Construcción del dataset final
- Análisis univariado y bivariado
- Visualización e interpretación de resultados

## Dataset utilizado

**Diabetes Health Indicators Dataset**

Fuente:  
https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset

El dataset contiene información relacionada con:

- Índice de masa corporal (BMI)
- Actividad física
- Tabaquismo
- Edad
- Hábitos de salud
- Enfermedades asociadas
- Estado de diabetes

## Objetivo del análisis

Identificar posibles relaciones entre variables asociadas a hábitos de vida y el estado de diabetes, con el fin de comprender patrones relevantes y preparar los datos para futuras etapas predictivas.
